# Deploy CCFoundry Dev Board

## Deploy And Open Dev Board

Your CCFoundry Agent Kit checkout is ready in Google Cloud Shell.

Click the button on the code block below to deploy CCFoundry Agent Dev Board.
The final line printed in the terminal is the Web UI URL.

```sh
bash deploy.sh && cat .ccfoundry-dev-board-url
```

If Cloud Shell asks you to choose a Google Cloud project, select the project you
want to use for this demo and run the same command again.

Open the printed Web UI URL to start using CCFoundry Agent Dev Board.
